Projetos

LinkFly

Encurtador de Links: Projeto desenvolvido para otimizar o compartilhamento de URLs longas, utilizando HTML, CSS, JavaScript no front-end e Node.js com Express.js no back-end. A aplicação permite que os usuários insiram links e gerem versões encurtadas, com redirecionamento eficiente e rápido. Focado em uma interface simples e funcional, o projeto também inclui práticas de manipulação de rotas e configuração de servidor básico para garantir a performance e a escalabilidade da aplicação.
